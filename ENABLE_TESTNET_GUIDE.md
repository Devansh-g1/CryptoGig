## üéØ Common Issues & Solutions

### Issue: "Show test networks" option not visible

**Solution**:
- Update MetaMask to latest version
- Go to: chrome://extensions
- Find MetaMask
- Click "Update"
- Restart browser

---

### Issue: Sepolia not in list after enabling

**Solution**:
- Use Method 2 (Add Manually)
- Or use Method 3 (Auto-add via platform)

---

### Issue: Can't switch to Sepolia

**Solution**:
1. Disconnect wallet from all sites
2. Close and reopen MetaMask
3. Try switching again
4. If still fails, remove and re-add Sepolia

---

### Issue: Balance shows 0 ETH on Sepolia

**Solution**:
- This is normal for new testnet
- Get test ETH from faucets (see above)
- You need ETH for gas fees

---

### Issue: Platform says "Wrong network"

**Solution**:
1. Check wallet shows "Sepolia" at top
2. If not, switch to Sepolia
3. Refresh platform page
4. Reconnect wallet
